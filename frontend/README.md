# Frontend веб-приложения для ведения личного бюджета

Клиентская часть реализована на React с использованием Webpack и React Router.

## Запуск

```bash
npm install
npm start
```

Приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Сборка production-версии

```bash
npm run build
```

## Основные разделы

- Регистрация
- Вход
- Главная
- Операции
- Планирование бюджета
- Настройки

## Подключение к backend

По умолчанию используется адрес:

```text
http://localhost:8000
```

При необходимости его можно изменить в `src/api/client.js`.
